Yandex.Turbo export module
=======================================

Note: Russian version of this documentation see below.

Yandex.Turbo module provides simple way to create nodes
export for Yandex search engine (https://yandex.ru/)

Features
========

* Node types export configuration
* Customizable sort order
* RSS title, description
* Provides allowed html tags filter

Installation
============

* Backup your database.

* Copy the complete 'yandex_turbo/' directory into the 'sites/all/modules/',
  'sites/default/modules' or 'sites/name_of_your_site/modules' folder of your Drupal setup.

* Enable the "Yandex.Turbo" module from the module administration page (Administer >> Site building >> Modules).


Configuration
=============

* Customize Yandex.Turbo RSS settings (Administer >> Configuration >> Web services >> Yandex Turbo)
  admin/config/services/yandex-turbo

* RSS link /yandex.turbo.rss

Яндекс.Турбо RSS
=======================================

Модуль позволяет быстро выбрать и выгрузить любые типы в RSS канал
для Яндекс.Турбо-страниц


Возможности
===========

* Настройка экспортируемых типов нод
* Настройка порядка сортировки
* RSS заголовок, описание
* Фильтр разрешенных HTML-тегов

Установка
=========

* Создайте резервную копию вашей базы данных.

* Скопируйте директорию 'yandex_turbo/' целиком в 'sites/all/modules/' или 'sites/default/modules'
или 'sites/имя_вашего_сайта/modules' директорию вашей Drupal установки.

* Включите модуль "Yandex.Turbo" на странице управления модулями (Управление >> Конструкция сайта >> Модули).


Настройка
=========

* Настройка Yandex.Turbo RSS (Управление  >> Настройка >> Веб сервисы >> Yandex Turbo)
admin/config/services/yandex-turbo

* Ссылка на выгрузку http://ваш_сайт/yandex.turbo.rss
